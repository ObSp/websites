JGame

Role: Creator & Maintainer

Description:

Since learning Java a few months ago, I've always disliked how annoying it is to create even a simple game. To fix this, I created the JGame framework. It uses all the building blocks included in the AWT and Swing libraries but provides a developer friendly API with a stable game loop, many different instances like images and rectangles and services that allow things such as tweening, raycasting, and detecting input. Many names and functionalities of this framework were abstracted from Roblox.
